yii2rbac
========

RBAC Module for developed by Yii2 

1. Here enclosed model,controller & view files only.

Task Details :
===============

Roles:
----------
• System Administrator
• SuperAdmin
• CompanyAdmin
• Manager
• Employee

Pages:
--------
User Frontend:
---------------
• Landing (redirect to login)
• Forgotten username (email validation)
• Forgotten password (reset token via email)
• User Profile Edit (incl. Password chagne)
Admin Backend:
------------------
• User Listing
• Edit Users
• Create Users
After Login:
-----------------
• Redirect user to Welcome Page based on Role (each role a controller)
